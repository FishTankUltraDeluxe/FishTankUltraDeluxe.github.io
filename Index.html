The Smiley Dictionary

2017

This is a snapshot of a layout project completed as part of my education at OCADU. We were provided with a set of copy which we were then required to arrange in a typographic layout using Adobe Indesign. We were required to utilize all formating that the source document included, and to communicate with our 'client' if we spotted any potential errors or isssues in the copy.
